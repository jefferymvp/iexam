语言模型对话- V3（推荐使用）
================

更新时间：2026.03.27 10:36

### **接口描述**

提供语言模型的对话能力，可通过model参数传入模型名称，部分模型支持深度思考或工具调用

深度思考：针对复杂问题，模拟人类思维过程，分步骤拆解问题并推导结论，支持多轮对话中延续推理逻辑

工具调用（Function Call）：支持通过Tool参数传入可用的工具列表及其描述，模型根据用户问题，识别意图，生成对这些工具的标准化调用请求，具体说明

### **接口信息**

| 请求方式 | POST                                                             |
| ---- | ---------------------------------------------------------------- |
| 请求地址 | https://jiutian.10086.cn/largemodel/moma/api/v3/chat/completions |

model使用：qwen3.5-397B-fp8

### **返回参数**

| **参数**  | **类型**          | **示例**      | **说明**    |
| ------- | --------------- | ----------- | --------- |
| model   | String          | jiutian-lan | 模型名称      |
| choices | Array of Choice | []          | 模型回复      |
| usage   | Object of Usage | {}          | token消耗统计 |

**Choice（流式）**

| **参数** | **类型**          | **示例**                             | **说明**   |
| ------ | --------------- | ---------------------------------- | -------- |
| delta  | Object of Delta | {"content": "","role":"assistant"} | 流式增量返回数据 |
| index  | Integer         | 0                                  | 索引标号     |

**Choice（非流式）**

| **参数**        | **类型**            | **示例** | **说明**   |
| ------------- | ----------------- | ------ | -------- |
| index         | Integer           | 0      | 索引标号     |
| message       | Object of Message | {}     | 对话回复消息内容 |
| finish_reason | String            | stop   | 结束原因     |

**Delta**

| **参数**            | **类型**            | **示例**    | **说明**                        |
| ----------------- | ----------------- | --------- | ----------------------------- |
| role              | String            | assistant | 对话角色，分为：system、user、assistant |
| content           | String            | You       | 增量回复内容                        |
| reasoning_content | String            | You       | 思考过程                          |
| tool_calls        | Array of ToolCall |           | 调用指令                          |

**ToolCall**

| **参数**   | **类型**                | **示例** | **说明**   |
| -------- | --------------------- | ------ | -------- |
| index    | Integer               | 0      | 索引标号     |
| function | Object of FuntionInfo | You    | 函数调用指令信息 |

**FuntionInfo**

| **参数**    | **类型** | **示例** | **说明** |
| --------- | ------ | ------ | ------ |
| name      | String |        | 函数名称   |
| arguments | String |        | 函数参数信息 |
